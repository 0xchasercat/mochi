# Terminal Demo UI Kit

A living recreation of the **mochi.js CLI / terminal experience** — the "stealth" surface of the brand.

## Components

- `TerminalWindow.jsx` — macOS-style window chrome with traffic lights and tab strip
- `Prompt.jsx` — the `mochi $` prompt
- `Output.jsx` — log lines, status chips, fingerprint reveals
- `MochiBanner.jsx` — ASCII / mascot splash shown on `mochi --hello`

## Entry

`index.html` — animated, click-thru CLI session that boots a stealth browser and prints fingerprint diagnostics.

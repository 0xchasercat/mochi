# mochi.js Docs Site UI Kit

A high-fidelity recreation of how the **mochi.js docs / marketing site** should look under this design system. No real codebase was provided — this is an interpretive build from the brand language.

## Components

- `Nav.jsx` — sticky top nav with wordmark mini, search, GitHub
- `Hero.jsx` — mascot + wordmark + headline + dual CTA
- `FeaturePillars.jsx` — five totem-emoji feature blocks
- `CodeShowcase.jsx` — stealth-surface code block w/ tabs
- `Footer.jsx` — minimal footer with wordmark

## Entry point

`index.html` — assembles all components into a single landing-page view.
